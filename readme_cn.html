UHD
(Universal Hardware Drive)

 

UHD(Universal Hardware Drive, 通用硬件驱动)是一套简洁高效的单片机底层驱动，它可以用于多种单片机，如：AVR, PIC10, PIC12, PIC16, PIC18, PIC24, MSP430, 8x51, STM8 等。使用相同的函数或者接口来驱动不同单片机的 GPIO, 定时器, 串口，中断和其它外设，让编程和程序移植变得轻松和简单。

授权许可

UHD 是开源项目，在大多数情况下可以免费使用。它使用了 MIT 授权。

Do not be evil.

 

特点

  • 使用简单。
  • 支持多种单片机和微处理器。
  • 在不同架构间使用相同的接口和函数。
  • 支持主流编译器。
  • 代码简洁高效。
  • 支持功能裁剪。
  • 可以和其它RTOS、库函数同时使用。

 

支持的单片机

  • PIC10 系列。使用 PIC10F32x 开发板 进行测试。
  • Atmega328p。使用Arduino Nano 3测试。

 

更多型号在逐步添加中。

 

使用方法

添加文件 uhd.c 和 uhd.h 到你的项目文件中，并添加文件的目录到项目的搜索路径中。

如果使用了功能裁剪，还需要添加（或新建）uhdcutdown.h文件，并在uhdcutdown.h文件中指定需要裁剪的功能模块，同时还需要在uhd.h中将宏 _UHD_CUTDOWN_ 设置为 1。

IO

 

 

 

 

 

定义IO

 

#define NAME  PORT, INDEX

 

 


NAME是 IO 的名称
PORT是 IO 所在端口的名称
INDEX是 IO 的序号
 

  例如

 

#define LED     A, 1
#define BUTTON  A, 3

 

 

 

设置输入输出

 

IO_dir(PIN, DIR);

 

 


PIN是上面定义的IO
  IO_INPUT代表输入
  IO_OUTPUT代表输出
 

  例如

 

IO_dir(LED, IO_OUTPUT);

 

 

 

输出高电平

 

IO_set(PIN);

  例如

 

IO_set(LED);

 

 

 

输出低电平

 

IO_clr(PIN);

  例如

 

IO_clr(LED);

 

 

 

输出翻转

 

IO_inv(PIN);

 

 


翻转输出状态。原来是高电平，那么就变为低电平；原来是低电平，就变为高电平。
 

  例如

 

IO_inv(LED);

 

 

 

读取IO电平

 

IO_in(PIN);

 

 


读取IO上的电平。

返回参数：

  IO_HIGH  高电平
  IO_LOW   低电平
 

  例如

 

IO_in(BUTTON);

 

 

 

设置数字功能

 

IO_digital(PIN);

 

 

 
允许IO的数字输入功能
 

  例如

 

IO_digital(BUTTON);

 

 

 

设置模拟功能

 

IO_analog(PIN)

 

 

 
允许IO的输入功能
 

  例如

 

IO_digital(POT);

 

 

 

设置上拉电阻

 

IO_pullup(PIN, PULLUP)

 

 


设置IO的内部上拉电阻。

PIN代表需要设置的IO，PULLUP代表上拉电阻状态。

  PULLUP_ENABLE  允许上拉电阻
  PULLUP_DISABLE 禁止上拉电阻
 

  例如

 

IO_pullup(BUTTON, PULLUP_ENABLE);

 

 

 

TIMER

 

 

 

 

 

设置模式

 

TMR_init(uint8_t no, uint16_t interval, void (* isr)()); 

 

 

 
no 代表定时器模块
  TMR_0  代表定时器0
  TMR_1  代表定时器1
  TMR_2  代表定时器2，以此类推。

interval代表定时器周期，单位是ms。如果interval是0，定时器将被禁止。

isr代表用户的中断服务程序。
 

  例如：

 

TMR_init(TMR_0, 1000, &myTMR_ISR);
