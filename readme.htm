UHD
(Universal Hardware Drive)

 

Universal Hardware Drive is a set of macro and function for microcontrollers, such as AVR, PIC10, PIC12, PIC16, PIC18, PIC24, MSP430, 8x51, STM8, etc. With same interface to drive microcontroller's GPIO, timer, uart and other peripherals.

Feature

  • Easy to use.
  • Use same interface/function between microcontrollers.
  • Small and efficient.
  • support cutdown to reduce.

 

Tested Microcontroller

  • PIC10F322. Test with Microchip PIC10F32x Development Board.
  • ATmega328P. Test with Arduino nano 3.

 

More device is being added.

 

Usage

Add files uhd.c and uhd.h to your project.

If you use cutdown feature, you need set macro _UHD_CUTDOWN_ to 1 in file uhd.h, then add file uhdcutdown.h and set cutdown macro in it.

IO

 

 

 

 

 

Define IO

 

#define NAME  PORT, INDEX

 

 


NAME: Name of a IO
PORT: IO's PORT of MCU
INDEX: IO's PORT index
 

  example

 

#define LED     A, 1
#define BUTTON  A, 3
