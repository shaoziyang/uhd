

Usage

For 
1. copy files uhd.c and uhd.h to your project folder.
2. add uhd.c and uhd.h to you project.
3. add uhd.h path to your project.
4. include "uhd.h" in your file.
5. now you can using uhd, please see demos for more.

For arduino
1. import library uhd.zip to you arduino IDE.
2. include <uhd.h> in your sketch file.
3. start to using uhd in your design.
