This folder contain several demos for UHD. All demos routines has been tested in a Arduino Nano 3 board.

Notes:
* Before compile the demo file, you need install WinAVR or GNU Toolchain for Atmel AVR8.
* Most demos may run under Labcenter proteus8, and use avrdude download hex to arduino board.

arduinoNano_demo.zip    For Arduino IDE, tested in Arduino IDE 1.05/1.57.
                        Before compile the demo, import arduino_library first.
                        
AS4_demo.zip            For old Avr Studio 4 IDE, it is easy to use for avr microcontroller.

AS7_demo.zip            For new ATMEL Studio 7 IDE.
